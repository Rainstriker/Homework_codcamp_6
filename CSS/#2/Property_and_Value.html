Property VS Value

CSS Code Example

body {

font-size: 14px;
color: navy;
}

Property คือ คุณสมบัติที่ใช้กำหนดรูปแบบ เช่น: font-size, color

Value คือ ค่าของProperty เช่น: 14px, navy

Precamp Property

Content Border Padding Margin Element
color , size , font , text, list-style-type, Height, Width, background border, table padding (top right buttom left) margin (top right buttom left) display, Height, Width, Position, Float, Align, Flex, Grid

Value Unit

Absolute Lengths

Unit Description
cm centimeters
mm millimeters
in inches (1in = 96px = 2.54cm)
px pixels (1px = 1/96th of 1in)
pt points (1pt = 1/72 of 1in)
pc picas (1pc = 12 pt)

Relative Lengths

Unit Description
em Relative to the font-size of the element (2em means 2 times the size of the current font)
ex Relative to the x-height of the current font (rarely used)
ch Relative to the width of the "0" (zero)
rem Relative to font-size of the root element
vw Relative to 1% of the width of the viewport
vmin Relative to 1% of viewport's smaller dimension
vmax Relative to 1% of viewport's larger dimension
% Relative to the parent element
