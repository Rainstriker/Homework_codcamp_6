CSS Combinators

  • Descendant Selector
    Combine selectors by using whitespace. It means select descendant element that consist all selected tags
    example: div p {background-color: yellow;}
  • Child Selector
    Combine selectors by using ">". It means select child element that under parent tag
    example: div > p {background-color: yellow;}
  • Adjacent Sibling Selector
    Combine selectors by using "+". It means select adjacent element to the first one
    example:div + p {background-color: yellow;}
  • General Sibling Selector
    Combine selectors by using "~". It means select all sibling elements that after the first tag
    example:div ~ p {background-color: yellow;}
